# skills/

This directory is reserved for **development and architecture skills** — skills that help when working on the openbrain system itself (schema migrations, smoke-test debugging, ingestion pipeline work, etc.).

**User-facing skills** (auto-capture, panning-for-gold, and any future vault workflow skills) live in [`packages/pi-open-brain/skills/`](../packages/pi-open-brain/skills/) and are distributed as part of the `pi-open-brain` package.
